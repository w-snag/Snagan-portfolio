Glowing

I created the glowing feature around a picture using CSS by wrapping the image in a styled container and applying a box-shadow effect with a bright color. To make the glow dynamic, I used @keyframes to animate the shadow, creating a pulsing light effect that smoothly expands and fades in a loop. This technique adds a modern and eye-catching visual enhancement to the image.

Bouncing

I implemented the bouncing image effect using CSS animations by defining a @keyframes rule that moves the image up and down using the transform: translateY() property. I applied this animation on hover, so the image only starts bouncing when the user interacts with it. This subtle movement adds interactivity and liveliness to the design, making the image stand out without overwhelming the page.
